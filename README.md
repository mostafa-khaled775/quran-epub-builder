# Quran EPUB Builder

## Usage

### Command-line Arguments

- `-n <narration>`: Specify the narration to download and process. If not provided, defaults to "Hafs".

### Valid Narrations

- `Hafs`
- `Warsh`
- `Shubah`
- `Qaloun`
- `Duri`
- `Susi`

### Example Usage

```sh
./build-epub.sh -n Warsh
```
